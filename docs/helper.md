# Хелперы (Helper)

## Назначение
- **Цель** Предоставить вспомогательные функции для отладки и визуализации данных.

## Архитектура
- **Файл** `application/helpers/helper_helper.php` содержит функцию `prettyDump()`.
- **Назначение** функция выводит содержимое переменных в читаемом формате, опционально показывает стек вызовов и останавливает выполнение.

## Использование `prettyDump()`
- **Сигнатура** `prettyDump($data = null, $die = false, $showStack = false)`.
- **Условия** работает только в локальной среде (`SERVER_ADDR` 127.0.0.1 и т.п.).
- **Параметры**
  - `$data` — выводимая переменная.
  - `$die` — завершить выполнение после вывода.
  - `$showStack` — показать стек вызовов.

## Расширение
- **Дополнительные утилиты** можно добавить в этот файл, следуя стилю CodeIgniter.
- **Логи** интеграция с мониторами ошибок.
