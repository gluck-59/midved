# Авторизация (Login)

## Назначение
- **Цель** Предоставить форму входа и перенаправление авторизованных пользователей в систему.

## Архитектура
- **Контроллер** `application/controllers/Login.php` обрабатывает маршрут `/login`.
- **Модель** `application/models/UserModel.php` используется для проверки учетных данных.
- **Представление** `application/views/login.php` (или `login.php` в корне) содержит форму входа.

## Поток
- **Ввод** пользователь отправляет `user`, `password`.
- **Проверка** контроллер передаёт данные в `UserModel::isAuth()`; при успехе происходит редирект на `/welcome`.
- **Хранение** успешная аутентификация устанавливает `$_SESSION['user_id']`.

## Расширение функциональности
- **Валидация** добавить сообщения об ошибках.
- **Защита** ограничение по IP, Captcha, блокировка при множественных попытках.
- **Регистрация/сброс пароля** через дополнительные маршруты.
