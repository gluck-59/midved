# Главная страница (Welcome)

## Назначение
- **Цель** Предоставить обзор состояния CRM и быстрый доступ к ключевым действиям.

## Архитектура
- **Контроллер** `application/controllers/Welcome.php` подготавливает данные для главной страницы, включая клиентов, оборудование, заявки и отчёты.
- **Представление** `application/views/main.php` содержит основную разметку и кнопки быстрого доступа.
- **JavaScript** `application/js/script.js` управляет модальными окнами и действиями, доступными с главной.

## Основные разделы
- **Модальные окна** вызываются для создания клиентов, оборудования, заявок (`modals.php`).
- **Сводные блоки** показывают ключевые метрики и быстрые ссылки (подробности в `main.php`).
- **Скрипты** инициализируют автозаполнение списков, запуск авторазноски.

## Расширение функциональности
- **Виджеты** добавить дополнительные отчеты, диаграммы.
- **Персонализация** отображать данные конкретного пользователя.
- **Уведомления** интегрировать с задачами/напоминаниями.
