# Календарь (Calendar)

## Назначение
- **Цель** Отображать календарный интерфейс (на данный момент простая заглушка).

## Архитектура
- **Контроллер** `application/controllers/Calendar.php` обслуживает `/calendar`, загружая `header`, `calendar`, `footer`.
- **Представление** `application/views/calendar.php` (не просмотрено) должно содержать UI календаря.

## Состояние
- **Функционал** минимальный, без логики и моделей.
- **Расширение** требует внедрения модели или интеграции с заявками/платежами.

## Рекомендации по развитию
- **Подключить данные** о заявках/платежах.
- **Добавить API** для фильтрации событий.
- **Интегрировать** с существующими сущностями (клиенты, оборудование).
