# Основное представление (Main)

## Назначение
- **Цель** Служить центральной страницей интерфейса, объединяющей карточки клиентов, оборудования, заявок и быстрые действия.

## Архитектура
- **Представление** `application/views/main.php` отображает виджеты, кнопку вызова модальных окон, сведения об отчетах.
- **Контроллер** `Welcome.php` передает необходимые данные (`customers`, `equipment`, `requests`).
- **Модальные окна** подключаются через `application/views/modals.php`.
- **JavaScript** `application/js/script.js` реализует действия из главного экрана (создание сущностей, авторазноска платежей, обновление селектов).

## Основное содержание
- **Карточки** показывают статистику по клиентам, оборудованию, заявкам, платежам.
- **Действия** кнопки для открытия модальных окон создания сущностей.
- **Отчеты** ссылки на отчёты из `Report`.

## Расширение функциональности
- **Виджеты** интеграция дополнительной аналитики.
- **Настройки пользователя** кастомизация отображаемых блоков.
- **Уведомления** подключение систем задач/напоминаний.
